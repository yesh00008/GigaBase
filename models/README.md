# Models

This directory contains transformer architectures, model implementations, and pre-trained weights.

## 🤖 Available Models

Browse the folders below to find models. Each model includes:
- Model card with detailed documentation (`.md` file)
- Implementation code
- Links to pre-trained weights
- Training details and hyperparameters
- Performance benchmarks
- Usage examples

## 📝 Contributing a Model

To contribute a model:

1. Create a new folder: `models/your_model_name/`
2. Add implementation code and configs
3. Create model card: `your_model_name.md` (use [TEMPLATE.md](TEMPLATE.md))
4. Include:
   - Architecture details
   - Training information
   - Performance benchmarks
   - Usage examples
   - Links to weights
   - License information

See [TEMPLATE.md](TEMPLATE.md) for the full model card template.

## 🎯 Model Card Best Practices

When creating a model card:
- Be thorough with architecture specifications
- Include training hyperparameters
- Document any limitations or biases
- Provide clear usage examples
- Link to pre-trained weights
- Specify hardware requirements
- Include benchmark results
- Add ethical considerations

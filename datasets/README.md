# Datasets

This directory contains curated datasets for LLM training, fine-tuning, and evaluation.

## 📊 Available Datasets

Browse the folders below to find datasets. Each dataset includes:
- Comprehensive documentation (`.md` file)
- Data files or links to download
- License and usage information
- Preprocessing details
- Usage examples

## 📝 Contributing a Dataset

To contribute a dataset:

1. Create a new folder: `datasets/your_dataset_name/`
2. Add your data files or provide download links
3. Create documentation: `your_dataset_name.md` (use [TEMPLATE.md](TEMPLATE.md))
4. Include:
   - Overview and description
   - Keywords/tags
   - Dataset statistics
   - Format and structure
   - Preprocessing steps
   - Usage examples
   - License information

See [TEMPLATE.md](TEMPLATE.md) for the full documentation template.
